## 1. プロジェクト概要

### 1.1 目的

TOEIC学習の記録・可視化・継続をサポートするWebアプリケーション。学習セッションの記録、スコア推移の把握、語彙管理を一元化し、学習習慣の定着を促進する。

### 1.2 背景

- TOEIC受験に向けた学習管理ツールの自作
- フルスタックポートフォリオとして採用活動にも活用

### 1.3 技術スタック

| レイヤー | 技術 |
|----------|------|
| フロントエンド | React + TypeScript + Vite |
| スタイリング | Tailwind CSS |
| データフェッチ | TanStack Query |
| グラフ | Recharts |
| バックエンド | Spring Boot 3 (Java 17) |
| ORM | Spring Data JPA |
| DB | MySQL |
| ビルド | Maven |

## 4. 機能詳細

### F-01: 学習セッション記録

**入力項目**

| フィールド | 型 | 必須 | バリデーション |
|-----------|-----|------|----------------|
| 学習日 | date | ✅ | 未来日付不可 |
| 学習時間（分） | integer | ✅ | 1〜480 |
| カテゴリ | enum | ✅ | 下記参照 |
| メモ | string | ❌ | 最大500文字 |

**カテゴリ一覧**

- `LISTENING` リスニング
- `READING` リーディング
- `VOCABULARY` 語彙
- `GRAMMAR` 文法
- `MOCK_EXAM` 模擬試験
- `OTHER` その他

---

### F-02: スコア記録

**入力項目**

| フィールド | 型 | 必須 | バリデーション |
|-----------|-----|------|----------------|
| 受験日 | date | ✅ | 未来日付不可 |
| Total スコア | integer | ✅ | 10〜990 (5の倍数) |
| Listening スコア | integer | ✅ | 5〜495 (5の倍数) |
| Reading スコア | integer | ✅ | 5〜495 (5の倍数) |
| 備考 | string | ❌ | 最大200文字 |

**制約**

- Listening + Reading = Total であること

---

### F-05: アクティビティヒートマップ

- 過去1年分（52週）を表示
- 学習時間に応じた4段階の色強度
- ホバーで日付・学習時間を表示

---

### F-07: 語彙管理

**入力項目**

| フィールド | 型 | 必須 |
|-----------|-----|------|
| 単語/フレーズ | string | ✅ |
| 意味 | string | ✅ |
| 例文 | string | ❌ |
| 習熟度 | enum (1〜5) | ✅ |
| タグ | string[] | ❌ |
